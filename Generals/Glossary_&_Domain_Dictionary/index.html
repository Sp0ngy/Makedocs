    2. be descriptive, but not too wordy
    3. avoid names using single character

    8. UpperCaseCamelCase MyExampleClass
    9. Exceptions should end in Error like MyExampleClassError

    11. Lowercase with underscore my_var, run_method

    14. Fully capitalized, separated by underscore MY_GLOBAL_CONSTANT
